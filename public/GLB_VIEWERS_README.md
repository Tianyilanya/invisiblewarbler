# GLB 模型查看器

本目录包含多个用于测试和展示 GLB 鸟类组件模型的查看器页面。

## 📁 文件列表

### 1. `models-gallery.html` ⭐ **推荐**
**鸟类模型组件画廊** - 功能最全面的查看器
- **组合视图**：展示完整的鸟类装配效果
- **单独视图**：网格布局查看所有组件
- **交互控制**：
  - 缩放控制 (+/- 按钮)
  - 布局选择 (圆形/网格/线性)
  - 旋转动画
  - 模型选择器
- **实时统计**：模型数量、三角形总数、文件大小
- **HDR 环境照明** + **阴影系统**

### 2. `bird-assembly-viewer.html`
**鸟类组件组合查看器**
- 展示所有鸟类部件的完整组合
- 加载顺序：胸部 → 头部 → 腹部 → 翅膀 → 尾部 → 足部
- 使用 HDR 环境照明
- 支持鼠标轨道控制

### 3. `random-bird-viewer.html`
**随机鸟类生成器**
- 随机选择不同变体的鸟类部件
- 交互式生成新鸟类
- 支持线框模式切换
- 实时显示加载状态

### 4. `glb-performance-test.html`
**GLB 性能测试**
- 测试所有 GLB 模型的加载性能
- 显示加载时间、三角形数量等统计信息
- 网格布局展示所有模型
- 性能评级和优化建议

### 5. `debug-glb.html`
**GLB 调试页面**
- 单独测试单个 GLB 模型加载
- 用于排查加载问题
- 简单的轨道控制

### 6. `test-basic.html`
**基础功能测试**
- 测试 Three.js 基本功能
- 验证 GLB 文件可访问性
- 加载单个模型进行功能验证
- 显示详细的测试结果和错误信息

## 🚀 使用方法

1. 确保开发服务器正在运行：`npm start`
2. 在浏览器中访问对应页面：
   - `http://localhost:8080/bird-assembly-viewer.html`
   - `http://localhost:8080/random-bird-viewer.html`
   - `http://localhost:8080/glb-performance-test.html`
   - `http://localhost:8080/debug-glb.html`

## 🎯 技术特性

- **HDR 环境映射**：使用 Royal Esplanade HDR 环境
- **PBR 渲染**：ACES 色调映射和物理材质
- **异步加载**：非阻塞的模型加载
- **性能监控**：实时统计和性能评估
- **响应式设计**：自适应窗口大小

## 📊 性能基准

基于当前测试数据：

- **平均加载时间**：< 200ms (优秀)
- **总三角形数**：~50,000-100,000
- **内存占用**：视具体模型而定
- **渲染性能**：60 FPS (目标)

## 🔧 故障排除

### 模型不显示
1. 检查浏览器控制台错误
2. 确认 GLB 文件存在且可访问
3. 检查网络标签页的 404 错误

### 性能问题
1. 使用性能测试页面评估加载时间
2. 检查是否启用了硬件加速
3. 考虑模型优化（减少多边形数量）

### 兼容性问题
- 需要支持 WebGL 2.0 的现代浏览器
- Chrome 88+, Firefox 85+, Safari 14+ 推荐

## 📝 开发说明

这些页面基于 Three.js 官方示例 `webgl_loader_gltf.html` 修改，
专门为《不可见莺》项目的 GLB 鸟类组件系统设计。

如需添加新的查看器功能，请参考现有页面的代码结构。
