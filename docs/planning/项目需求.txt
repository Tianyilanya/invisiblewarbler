项目名称：不可见莺（The Invisible Warbler）【新版】

================================================================================

项目概述

《不可见莺》是一个以GLB模型组件拼贴与动态鸟类生态模拟为核心的3D交互艺术项目。作品以"观鸟"这一自然观察行为为隐喻，通过抽象森林场景与性格鸟类生态，探讨艺术生成、生态平衡与算法生命的艺术命题。

在当代艺术语境中，"观鸟"不仅是一种自然观察行为，更是对碎片化观看经验的隐喻。观察者通过树叶的遮挡，只能捕捉到局部图像，需要依靠想象力将碎片拼合成完整形象。同样，面对当代艺术的各种"主义"，观众也需要在理论碎片中构建意义结构。

然而，不同于传统观鸟的被动性，本项目通过动态生成的鸟类生态，让观察者成为森林生态的一部分。鸟类具有不同的性格（观察者、巡逻者、旅行者、活跃者），在抽象森林中展现多样化的行为模式，创造出充满生命力的算法生态系统。

研究目的

探索算法生成艺术中的"生命"模拟机制，以及生态平衡在数字艺术中的表现形式。

通过性格鸟类的行为模式，探讨人工智能生成内容的情感化与拟人化表达。

模拟森林生态系统，通过组件化鸟类生成揭示数字生命的组合可能性与多样性。

反思抽象艺术与具象表达之间的张力，探讨线条、点云等抽象几何体在叙事中的表现力。

呈现项目

一个动态鸟类生态观察网页，包含以下核心体验：

1. 抽象森林场景：由线条树干和点云树冠组成的极简抽象森林，每棵树都有独特的形态，营造出充满艺术感的生态氛围。

2. GLB组件鸟类系统：鸟类由GLB模型部件（头部、胸部、腹部、翅膀、尾部、足部）自由组合生成，支持部件的动态加载和随机组装。

3. 性格鸟类生态：四种性格鸟类（观察者、巡逻者、旅行者、活跃者）在森林中展现不同的行为模式：
   - 观察者：主要悬停在树枝上，偶尔在附近巡逻
   - 巡逻者：持续在树木周围环形飞行，偶尔短暂悬停
   - 旅行者：频繁在不同树木之间穿梭飞行，偶尔悬停休息
   - 活跃者：各种行为频繁切换，最为活泼

4. 动态生态平衡：每5秒生成1-3只新鸟类，每只鸟类存活30秒，场景中最多同时存在5只鸟类。鸟类会在3-10秒后根据性格概率切换行为模式。

5. 实时监控界面：显示当前森林中各种鸟类性格的数量统计，用户可以切换鸟类定位模式（场景中心/随机树上）。

核心功能需求

【已实现功能】

1. 场景构建
   - 抽象森林生成：15棵随机分布的线条树木 + 点云树冠
   - 动态光照系统：方向光 + 环境光 + 雾效
   - 轨道控制器：自由视角观察

2. GLB组件系统
   - 文件夹结构：bird_components/{head,chest,belly,wing,tail,foot}
   - 自动部件发现：启动时扫描所有可用GLB文件
   - 组件缓存：加载结果缓存，避免重复加载

3. 鸟类生态系统
   - 性格分配：新鸟类随机分配四种性格之一
   - 行为状态机：hover/patrol/travel三种行为状态
   - 路径计算：悬停位置、环形巡逻、弧形穿梭
   - 生命周期：30秒自动消亡，内存自动清理

4. 交互控制
   - UI控制面板：鸟类定位模式切换
   - 统计面板：实时显示各种鸟类数量
   - 响应式设计：窗口大小自适应

【规划功能】

5. 用户交互系统（待开发）
   - 鼠标点击：观察和收集鸟类部件
   - 碎片胶卷：临时存储收集的部件
   - 合成机制：收集足够部件后触发组合

6. AI风格分析（待开发）
   - 自动分析：基于组合特征的艺术风格识别
   - 主义标签：生成相应的艺术主义名称
   - 解释文本：提供风格阐释和理论背景

7. 图鉴保存系统（待开发）
   - 作品保存：将组合结果保存到本地存储
   - 图鉴页面：浏览所有保存的作品
   - 3D预览：可旋转查看保存的鸟类雕塑

技术规格

- 平台：现代浏览器（Chrome 80+, Firefox 75+, Safari 13+）
- 3D引擎：Three.js r150+
- 模型格式：GLB/GLTF
- 几何体：线条(Line) + 点云(Points)抽象表达
- 性能目标：60 FPS稳定运行，内存占用 < 100MB
- 自适应：桌面端优先，移动端基础支持

用户体验目标

1. 沉浸式观察：抽象森林营造艺术氛围，让用户专注于鸟类行为生态
2. 发现乐趣：不同性格鸟类的行为模式带来持续的观察兴趣
3. 生态平衡：动态生成与清理创造生命的流动感
4. 艺术表达：线条与点云的抽象美学，符合艺术项目定位
5. 技术流畅：无卡顿的动画和响应，保证用户体验质量

项目价值

1. 技术创新：GLB组件化系统的灵活组合机制
2. 艺术表达：抽象几何体与算法生命的有机结合
3. 交互设计：性格系统创造丰富的行为多样性
4. 哲学思考：探讨数字生态与算法生命的可能性
5. 教育意义：通过鸟类行为理解生态平衡原理
