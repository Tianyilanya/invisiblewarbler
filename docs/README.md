# The Invisible Warbler Project Documentation

The Invisible Warbler is a three-dimensional interactive art web project that uses "bird watching" as a metaphor. Through web 3D interaction, GLB model component collage, and dynamic bird ecological simulation, it explores the artistic propositions of art fragments, generative mechanisms, and ecological balance.

Users act as "observers" in an abstract forest scene, experiencing a bird ecosystem composed of GLB model components, observing the behavioral patterns of birds with different personalities, and feeling the vitality of life generated by algorithms.

## ğŸš€ Quick Start

### Windows Users (Recommended)
1. Ensure Node.js and npm are installed
2. Double-click the `Invisible_Warbler.bat` file in the root directory
3. The batch file will automatically:
   - Start the local development server
   - webpack-dev-server will automatically open the app in your default browser
   - Server runs at http://localhost:8080

### Manual Startup
If the batch file doesn't work properly, you can start manually:
```bash
# Install dependencies (first run only)
npm install

# Start development server
npm start
```
Then manually visit http://localhost:8080 in your browser

### Stop the Server
Press `Ctrl+C` in the terminal window where the batch file is running to stop the server
